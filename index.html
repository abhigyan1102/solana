Level 1 · Forge the Rules

Register Your Agent. Write Its Destiny.

A flexible policy management system: register AI agents using PDAs and define granular, per-agent rules in seconds.

solanaguard — policy.forge
// derive the agent's on-chain identity
let agent_pda = Pubkey::find_program_address(
  [b"agent", owner.key(), b"trader_01"], program_id);

// carve the rules into the chain
register_agent(agent_pda);
set_policy(Policy {
  per_tx_limit:  5 SOL,
  daily_limit:   50 SOL,  // auto-resets every 24h
  allowlist:     [jupiter, orca, marginfi],
  kill_switch:   armed,
});  // ✓ sealed on-chain — nobody can unwrite this
