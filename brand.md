# Brand - SolanaGuard

Status: active

SolanaGuard is a premium dark Web3 security dashboard for AI-controlled Solana wallets. The product should feel like a Guardian command center inside a futuristic on-chain fortress: tactical, luminous, and high trust.

## Palette

- Void: `#030712`
- Fortress: `#07111f`
- Glass: `rgba(13, 24, 45, 0.72)`
- Solana Violet: `#9945ff`
- Solana Green: `#14f195`
- Neon Cyan: `#67e8f9`
- Warning Amber: `#fbbf24`
- Danger Red: `#fb7185`
- Text Primary: `#edf7ff`
- Text Muted: `#9fb0c7`

## Typography

- Display: Orbitron
- Body: Manrope
- Mono: IBM Plex Mono

## Voice

Concise, protective, tactical. Avoid hype; make risk decisions feel inspectable and controlled.
